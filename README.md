# Beanstalk - Child Growth Tracker Website

Marketing landing page for [Beanstalk - Child Growth Tracker](https://apps.apple.com/us/app/sprout-height-tracker/id6758862095), an iOS app for tracking your child's height and weight with WHO & CDC percentile growth charts.

## Live Site

[sudhanne-maker.github.io/sprout-child-growth-tracker-website](https://sudhanne-maker.github.io/sprout-child-growth-tracker-website/)

## Structure

```
├── index.html          # Single-page landing site
├── css/
│   └── style.css       # Responsive styles
├── images/
│   ├── app-icon.png    # App icon
│   ├── app-store-badge.svg
│   └── screenshot-*.png
└── README.md
```

## Development

Static HTML + CSS. No build tools required. Open `index.html` in a browser to preview.

## Deployment

Deployed via GitHub Pages from the `main` branch root.
